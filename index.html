Personas & Storyboarding

Part 1: Preparation

1) Select an accessible interface.

The selected interface is self checkout kiosk at CVS.

2) Prepare some questions for users regarding their experiences with the interface.

  1. What aspect of the interface do you like?
  2. What aspect of the interface do you dislike?
  3. What do you think is the most important feature that should be added?
  4. Which feature is least important to you?
  5. Is there any feature at the checkout kiosk that confuses you? If yes, which?
  6. What do you think of the layout of the checkout kiosk?
  7. Why did you choose self-checkout instead of manual checkout?
  8. Is there anything surprising or unexpected about the checkout kiosk?
  9. If you could improve one feature about the checkout kiosk, what would it be and why?
  10. What is your favorite payment method?

3) Create a sketch and write a brief description of the interface.

Part 2: Recording Observations

1) Write a few sentences or bullets describing the key observations you gained from observing your users.

  • Users like language selections, coupon scanning and various payment options about the check-out kiosk.
  • The most important feature that should be added is a Restart button. When a user scans an item but doesn't want it, or when the check-out machine is frozen, the user can restart the check-out by touching the Restart button.
  • Customers choose self-checkout instead of manual checkout for the following reasons. They like self-service; There is a line at the manual checkout; They don’t want to talk with others.
  • As for their favorite payment method, two prefer credit cards and the other one prefers Paypal.
  • One feature that should be improved is the receipt option. Currently, users only have two options: no receipt and print receipt. A third option to send electronic receipts should be added.
  • When a user used the check-out kiosk the first time, she felt confused about the coupon scanning

2) List your questions in a numbered list and summarize the responses of your interviewed users using up to 5 bullet points.
